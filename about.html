About the store

Designed as a realistic ecommerce demo, not a template shell.

Northstar Goods is a concept storefront built to showcase a complete shopping journey using only HTML, CSS, JavaScript, and localStorage.

What it includes

Pages, interactions, and content you can actually explore.

The layout includes a homepage, catalog, cart, about page, and support page. The cart, quantity controls, and support submissions persist locally for a more complete demo.

Visual language

Warm gradients, expressive type, and layered cards give the site a distinct look.

Browser state

Selected products, cart counts, and submitted tickets live in the browser only.

Frontend focus

No backend, no database, and no external data source are required to use the site.

Build philosophy

Simple enough to understand, polished enough to feel real.

Clear structure

Pages are split by intent so the experience feels like a true storefront.

Useful interactions

Cart and support actions are functional, not just decorative.

Local persistence

State survives refreshes, which makes the demo feel complete without a server.
