# full-frontend-ecommerce-website

A frontend-only ecommerce demo built with plain HTML, CSS, and JavaScript.

## Pages

- Home landing page with featured products and store highlights.
- Shop page with search, sorting, category filters, and add-to-cart actions.
- Cart page with quantity controls, subtotal, shipping, and simulated checkout.
- About page with the project story and build philosophy.
- Support page with a local-only support form and saved request history.

## Features

- Cart state persisted in `localStorage`.
- Support requests saved in `localStorage`.
- No backend, API, or database required.
- Responsive layout for desktop and mobile.

## Run It

Open the project in a browser through any static file server. For example:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.
